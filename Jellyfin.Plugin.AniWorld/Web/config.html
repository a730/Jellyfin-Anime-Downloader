AniWorld Downloader Settings

General

Maximum number of simultaneous downloads (1-5). Lower values are recommended to avoid rate limiting.
Number of times to retry a failed download (0 = no retries). Uses exponential backoff between attempts and will try alternative providers on failure.
Automatically trigger a Jellyfin library scan when a download completes. This makes new episodes appear in your library immediately.
Allow non-admin users to access the AniWorld Downloader UI via the sidebar. Requires the File Transformation plugin (recommended) or falls back to direct index.html modification. A server restart is required after changing this setting.
Block new downloads while allowing the current queue to finish. Users will see a maintenance message and download buttons will be disabled.
Message displayed to users when maintenance mode is active.
HTTP proxy for all network requests and downloads (e.g. http://proxy:8080 or socks5://proxy:1080). Leave empty to connect directly without a proxy. A server restart is required after changing this setting.
Optional dedicated base directory for movie downloads across all sources. Leave empty to use the normal site/language download paths.
The language the user requested is tried first. If it fails, it tries the selected priority, skipping the one which failed.
